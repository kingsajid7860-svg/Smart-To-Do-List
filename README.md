# Smart To-Do List

## Overview
The Smart To-Do List application is designed to help users manage their tasks efficiently. This application allows users to add, edit, and delete tasks easily, while also providing features like priority tagging and deadlines.

## Features
- **Add Tasks**: Users can add new tasks with details such as title, description, and due date.
- **Edit Tasks**: Tasks can be modified to update details.
- **Delete Tasks**: Users can remove tasks from their list.
- **Prioritization**: Users can categorize tasks based on priority levels (high, medium, low).
- **Visual Representation**: Tasks are displayed in a user-friendly interface that highlights due dates and priority levels.

## Getting Started
1. **Clone the Repository**: Clone the repository to your local machine using the following command:
   ```bash
   git clone https://github.com/kingsajid7860-svg/Smart-To-Do-List.git
   ```
2. **Install Dependencies**: Navigate to the project directory and install the required dependencies:
   ```bash
   cd Smart-To-Do-List
   npm install
   ```
3. **Run the Application**: Start the application using:
   ```bash
   npm start
   ```
   The application will be running on `http://localhost:3000`.
